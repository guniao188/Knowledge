# 常用服务器动作

```
for record in records:
    seller = env['product.supplierinfo'].create({
        'partner_id': 43,
        'product_tmpl_id': record.id,
        'min_qty': 1,
        'price': 0.0,  # 根据实际情况设定价格
        'sequence': 6,  # 设置优先级
    })
    # 将新创建的供应商信息设置为默认
    record.write({'seller_ids': [(4, seller.id)]})
```
