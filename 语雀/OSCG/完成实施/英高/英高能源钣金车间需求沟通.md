# 英高能源钣金车间需求沟通

![image](_附件/1730863581629-7c5b6cad-b7c7-4d70-97c9-61-2fb69ced161ebfc0.png)

总装车间需要领料：

1、原材料：总装车间找仓库要产品，若仓库没有产品，则跑出缺料的采购单

2、钣金车间半成品：总装车间找仓库要半成品，若仓库没有需要的半成品，会产生从钣金车间位置到仓库的入库单

2.1 钣金车间需要手动从仓库领料，进行加工后，将加工后的半成品进行入库。

2.2 若半成品入库后，则总装车间的钣金件入库单变成就绪状态。


仓库：1、钣金库；2、原材料库；3、撬装库；4、成品库


钣金车间仓管工作：

1、加工件钣金入库；

2、每月对钢板盘点，差异额=消耗钢板数量；

3、采购申请及采购入库；

4、钣金件发料


原材料库存管理：（需要区分产品的补货类型）

1、通用料：安全库存，根据库存消耗设置最低安全库存（手动补货/自动补货，前期采用仓管人员每天进行检查，手动选择补货）

2、按单购买产品：设置路线，订单有需求，则进行采购。


产品类别: 钣金、半成品、原材料、成品
