# Baseline

一、基础进销存

已完成:

1. 产品及产品相关参数收集及导入；
2. 香港公司供应商价格表、客户、供应商导入；
3. 进销存需要的核心功能增加:计算margin、添加销售单、采购单、入库、出库单字段增加及业务逻辑调整。目前进销存已跑通；

未完成：

1. 上海两家公司的供应商价格表、客户、供应商信息收集及导入；
2. 产品图片导入；


二、会计

已完成：

1. 香港公司会计科目表导入及配置科目自动产生凭证；
2. 分析账户设置；
3. 分析账户支持在订单行上选择，自动带到明细及后续单据；

未完成：

1. 上海两家公司科目表导入及自动凭证科目配置；
2. 合并多公司的财务报表；
3. 配置不同公司的财务报表取值逻辑；


三、市场

已完成:

1. 介绍CRM模块功能及解答问题；

未完成：

1. 介绍Social Marketing、Email Marketing模块；
2. 解决与网站的对接问题；
3. 其他需求待沟通；


四、人事

已完成：

1. 员工、部门信息收集与导入；
2. 假期类型设置；
3. 加班转调休功能开发及调整；
4. 解决按年分配假期控制使用天数功能；
5. 解决年底假期自动结转功能；

待完成：

1. 薪资计算功能；
2. 请假及费用报销审批流收集及设置
