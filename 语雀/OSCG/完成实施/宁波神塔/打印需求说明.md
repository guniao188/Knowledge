# 打印需求说明

1、Quotation Fromat 如下图所示：**---预计工时：3h**

![image](_附件/1755769805209-31be3a6d-586a-4b5b-a8a1-bc-d9e170d6b343fad9.png)

Sale Order增加字段：

logo Image 上传图片、

Quotation Company Name： 手动填写


2、PI Format

2.1	增加存储公司及对应银行信息的表单。可以维护多个公司及对应的银行信息**---预计工时：3h**

字段信息，均为手动填写字段:

Company Name

Swift code

Account Name

Account No

Bank name

Bank address


在订单上选择打款的 Company Account,PI中显示对应的Bank信息。


2.2 输出PI **---预计工时：9h**

销售订单增加字段：

Company Account:下拉框选择字段，选择2.1维护的信息中的一个，作为该订单的银行信息

P/I  No.：手动填写

Print Type:(Brand)PI、(Brand)PI with packing、(OEM)PI

Amount To Words:总金额从数字转成英文描述【系统自动计算】，界面不显示，仅供打印使用


公司增加字段：

Sign  Image：上传图片，用于输出PI，后续若有修改，直接换新的图片即可
