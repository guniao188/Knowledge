# 需求调研

**一、上线范围**

1. 目前第一个的上线公司抬头有几个？
2. 部门架构和职能架构
3. 目前是否使用ERP软件，使用的ERP软件是什么，后续是打算并用还是完全迁移
4. 预计上线时间及上线公司抬头
5. 请描述一下从销售员接到报价单后各个部门的工作流程


**二、销售**

1. 销售具体业务流程？销售的单价的单位是什么
2. 销售订单是否在Odoo中发送给客户确认？
3. 各公司各团队的业务员有多少？
4. 是否有价格表？销售员是否可以修改价格表？
5. 销售订单价格是否含税？
6. 销售订单如何开票，收款的流程和方式
7. 退货流程是怎么样的


**三、采购**

1. 采购整体流程是否包含审批，以销定采还是以采定销？
2. 是否有完整的供应商档案，供应商是否稳定
3. 是否有供应商价格表
4. 退货流程


**四、库存**

1. 目前是否有仓库？仓库和位置有哪些？其存储功能是什么？
2. 是否有专人来管理仓库？
3. 仓库入库是否涉及质检？
4. 现有库存是否精准？如不精准存在什么问题
5. 仓库出库发货是否涉及审批流程？如有，请提供审批流
6. 打包的费用是均摊到成本上还是直接进费用


**五、会计**

1. 目前是否使用财务系统，会计模块目前是如何考虑
2. 成本核算方式是先进先出还是平均成本
3. 如果涉及多公司，账套如何管理？是否合并账套？内外帐是否一套？
4. 目前的业务涉及哪几个外币币种？是否需要多语言？
