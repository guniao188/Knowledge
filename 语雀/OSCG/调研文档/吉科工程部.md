# 吉科工程部

1.核心业务流程

- 产品设计 → BOM制定 → 工艺参数设定 → 新品试制 → 标准化

2 关键问题/痛点

1. 新产品试制流程是否标准化？是否需小批量验证后再量产？  试产
2. BOM是否包含消耗定额？按照损耗率
3. 工艺路线是否细化到工序级？
4. 客户图纸或技术协议是否归档？变更时是否通知所有相关部门？  现在没有管理，微信通知生产车间
5. 是否积累历史工艺数据库？没有
6. 是否参与客户联合开发项目？知识产权如何归属？
7. 技术文件（作业指导书、控制计划）是否版本受控？现场是否使用最新版？
8. 是否有PLM系统？若无，图纸和BOM靠什么管理？（共享文件夹、纸质台账）没有
9. 受托委外工序
