# 吉科采购部

1 核心业务流程

- MRP生成采购需求 → 询价比价 → 采购订单 → 到货通知 → 入库 → 发票匹配

2 关键问题/痛点

1. 主要原材料（如钢坯、废钢、铁合金）的采购周期是多长？是否有安全库存策略？  有固定的供应商，优质供应商，
2. 采购单价的单位是什么？价格是否存在波动或最小起订量  吨采购，财务单位公斤。外地有最小起订量
3. 供应商是否提供批次？没维护批次
4. 是否对关键原料进行到厂复检？复检不合格如何处理？（退货/折价接收）
5. 包装材料（如打包带、护角、防锈纸）是否纳入BOM？是否由生产现场申领？  是
6. 是否存在“寄售采购”（供应商将物料存于我方仓库，用后结算）？如何管理库存权属？
7. 紧急采购是否跳过比价流程？事后是否补审批？有无审计风险？  需要财务审批
8. 是否跟踪供应商的交付准时率、质量合格率、服务响应速度？数据来源是什么？没有
