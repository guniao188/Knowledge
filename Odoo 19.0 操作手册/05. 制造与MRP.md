## 一、基础配置

### 1. 产品配置

#### 制造相关字段

| 字段 | 说明 |
|------|------|
| **产品类型** | 可库存产品 / 消耗品 / 服务 |
| **供应路线** | 购买 / 制造 / 按订单制造 (MTO) |
| **制造提前期** | 从下单到完工的工作日数 |
| **生产 BOM** | 制造该产品使用的物料清单 |
| **工艺路线** | 生产该产品经过的工序 |
| **成本方法** | 标准 / 平均 / FIFO |

#### 配置步骤

- 产品 → 库存页签 → 路线：勾选「制造」
- 产品 → 基本信息 → 制造提前期：设置生产周期

---

### 2. 物料清单 (BOM)

#### 字段解释

| 字段 | 说明 |
|------|------|
| **BOM 类型** | 制造 BOM / 套件 BOM / 分包 BOM |
| **产品** | 该 BOM 用于生产的产品 |
| **产品变体** | 特定变体适用的 BOM |
| **数量** | 该 BOM 生产多少单位成品 |
| **组件行** | 原材料、半成品、用量、损耗率 |
| **工艺路线** | 关联的工序流程 |
| **版本** | BOM 版本号（PLM管理） |

#### BOM 层级

```
产品A (成品)
├── 半成品B
│   ├── 原材料X
│   └── 原材料Y
├── 原材料Z
└── 原材料W
```

#### 配置步骤

1. **创建 BOM**
   - 制造 → 产品 → 物料清单 → 新建
   - 选择产品和数量
   - 添加组件行（直接材料/半成品）

2. **设置损耗**
   - 在组件行设置损耗率（如 5%）
   - 系统会自动放大需求数量

3. **多级 BOM**
   - 半成品也需建立 BOM
   - MRP 运算时会自动展开多级需求

#### 业务操作

- **BOM 复制**：相似产品的 BOM 可复制后修改
- **BOM 比较**：对比两个 BOM 的差异
- **BOM 成本**：根据组件成本自动计算标准成本

---

### 3. 工作中心

#### 字段解释

| 字段 | 说明 |
|------|------|
| **工作中心名称** | 如「装配线-A」、「CNC车间」 |
| **替代工作中心** | 产能不足时的替代选择 |
| **产能** | 并行处理数量（如同时加工 5 件） |
| **效率因子** | 实际效率 vs 标准效率 |
| **OEE 目标** | 设备综合效率目标值 |
| **成本/小时** | 该工作中心的小时费率 |
| **工作时间** | 班次日历 |

#### 配置步骤

- 制造 → 配置 → 工作中心 → 新建
- 设置产能、效率、成本、工作时间
- 关联设备（用于维护模块）

---

### 4. 工艺路线

#### 字段解释

| 字段 | 说明 |
|------|------|
| **工序** | 工艺步骤（如切割、焊接、喷涂、装配） |
| **工作中心** | 该工序执行的位置 |
| **设置时间** | 换线/调试时间 |
| **加工时间** | 单件加工时间 |
| **后处理时间** | 等待/冷却时间 |
| **指令** | 作业指导书 |

#### 配置步骤

- 制造 → 配置 → 工艺路线 → 新建
- 按顺序添加工序，指定工作中心和时间

## 二、MRP 运算与计划

### 1. MRP 自动运算

#### 功能概览

MRP (Material Requirements Planning) 根据需求预测、销售订单、库存水平和 BOM 自动计算需要生产/采购什么、何时、多少。

#### 触发条件

| 触发源 | 说明 |
|--------|------|
| **销售订单** | 确认 SO 后检查成品库存 |
| **安全库存低于** | 库存低于再订货点时触发 |
| **需求预测** | MPS 运算产生的计划订单 |
| **手动运行** | 手动执行 MRP 运算 |

#### 配置步骤

1. **启用 MRP**
   - 制造 → 配置 → 设置：启用「MRP」

2. **设置再订货点**
   - 产品 → 库存页签 → 最小/最大数量
   - 或手动创建「再订货规则」

3. **运行 MRP**
   - 制造 → 作业 → 运行 MRP
   - 或设置计划任务自动每日运行

#### 业务操作

- **MRP 异常**：查看缺料、延迟、产能不足等异常报告
- **计划订单**：确认 MRP 建议的制造/采购订单
- **批量确认**：一次性确认多个计划订单

---

### 2. 主生产计划 (MPS)

#### 功能概览

MPS (Master Production Schedule) 用于对关键成品进行中长期生产规划，平衡需求与产能。

#### 字段解释

| 字段 | 说明 |
|------|------|
| **预测需求** | 每月/每周的需求预测量 |
| **已确认订单** | 已接的销售订单量 |
| **计划产量** | MPS 建议的生产量 |
| **在手库存** | 当前库存 |
| **可用承诺 (ATP)** | 可承诺给新订单的数量 |

#### 配置步骤

1. **启用 MPS**
   - 制造 → 配置 → 设置：启用「主生产计划」

2. **设置 MPS 产品**
   - 在产品的库存页签勾选「在 MPS 中」

3. **录入预测**
   - MPS 视图中逐期录入需求预测

#### 业务操作

- **MPS 报表**：查看各期供需平衡情况
- **冲销规则**：销售订单自动冲销对应期的预测需求
- **冻结期**：设置近期计划不可自动变更

---

### 3. 需求预测

#### 配置步骤

- 制造 → 作业 → 预测
- 按产品/类别录入未来各期预测量
- 可基于历史销售数据自动生成预测建议

---

### 4. 补货报告

#### 功能概览

汇总所有需要补货的产品，展示建议的补货量和补货方式（采购/制造/调拨）。

#### 业务操作

- **查看补货**：制造 → 作业 → 补货
- **确认补货**：勾选建议行 → 确认 → 自动生成采购/制造订单
- **调整建议**：修改建议数量或补货方式后确认

## 三、制造订单

### 1. 创建制造订单

#### 字段解释

| 字段 | 说明 |
|------|------|
| **产品** | 要生产的产品 |
| **数量** | 生产数量 |
| **BOM** | 使用的物料清单 |
| **工艺路线** | 生产工序 |
| **计划开始/结束** | 排产时间 |
| **源位置** | 领料仓库 |
| **目标位置** | 完工入库仓库 |
| **状态** | 草稿 → 确认 → 进行中 → 完成 → 取消 |

#### 创建方式

- **手动创建**：制造 → 作业 → 制造订单 → 新建
- **MRP 自动生成**：MRP 运算后确认计划订单
- **从销售订单**：MTO 产品确认 SO 后自动生成

---

### 2. 领料与备料

#### 业务操作

- **自动领料**：MO 确认后自动生成领料单
- **备料单**：仓库按领料单准备物料
- **批量领料**：一次性领取全部物料
- **多次领料**：按工序分次领料
- **替代料**：主料不足时，经审批使用替代料

---

### 3. 生产执行

#### 业务操作

- **开始生产**：点击「开始」记录实际开始时间
- **工序报工**：每道工序完成后录入实际用时和产出
- **在制品跟踪**：查看各 MO 的生产进度
- **异常处理**：报废、返工、补料处理

---

### 4. 完工入库

#### 业务操作

- **生产完工**：点击「生产完成」
- **质检**：如配置了完工质检，需质检合格后入库
- **入库**：合格品入成品库，不合格品入废品或返工
- **成本结转**：系统自动结转材料成本+人工成本+制造费用

---

### 5. 生产成本核算

#### 成本构成

| 成本项 | 说明 |
|--------|------|
| **材料成本** | BOM 组件的领料成本 |
| **人工成本** | 工序的直接人工工时 × 人工费率 |
| **制造费用** | 工作中心的间接费用分摊 |

#### 配置步骤

- 设置 → 会计 → 成本方法：标准 / 实际
- 工作中心 → 成本页签：设置人工费率、间接费率

#### 业务操作

- **标准成本**：预设成本，差异单独分析
- **实际成本**：按实际领料和工时计算
- **成本分析**：查看各 MO 的实际成本 vs 标准成本差异

## 四、车间作业 (企业版)

### 1. 工单管理

#### 功能概览

将制造订单分解为多个工单 (Work Order)，按工序派发给工作中心执行。

#### 字段解释

| 字段 | 说明 |
|------|------|
| **工单号** | 系统生成 |
| **所属 MO** | 上级制造订单 |
| **工序** | 该工单对应的工序 |
| **工作中心** | 执行地点 |
| **计划时间** | 排产开始/结束 |
| **实际时间** | 实际开始/结束 |
| **数量** | 计划/实际产出数量 |

#### 业务操作

- **工单派发**：按工作中心查看待处理工单列表
- **优先级调整**：拖拽调整工单执行顺序
- **状态跟踪**：待处理 → 准备中 → 进行中 → 完成

---

### 2. 时间跟踪

#### 业务操作

- **计时器**：工单开始/暂停/结束计时
- **工时录入**：手动录入实际工时
- **效率分析**：实际用时 vs 标准用时的效率比

---

### 3. 设备状态

#### 业务操作

- **OEE 看板**：设备综合效率实时看板
- **状态切换**：运行 / 待机 / 故障 / 维护
- **报警**：设备异常时自动报警

## 五、高级功能

### 1. 外协加工

#### 功能概览

将部分工序外包给外部加工厂，Odoo 管理外协发料、收货、结算。

#### 配置步骤

1. **配置外协 BOM**
   - BOM 类型选择「分包」
   - 指定外协供应商

2. **外协价格**
   - 在 BOM 行或供应商价格表中维护加工费

#### 业务操作

- **外协订单**：自动生成外协 PO，发原材料给供应商
- **外协收货**：收到加工后的半成品入库
- **加工费结算**：按外协 PO 结算加工费

---

### 2. 副产品与联产品

#### 功能概览

生产过程中除了主产品外，还产生副产品（有价值的）或联产品（同等价值的产品）。

#### 配置步骤

- BOM 中添加「副产品」行
- 设置副产品的产出比例和成本分摊比例

---

### 3. 套件与拆解

#### 套件 (Kit)

- 销售时显示为一个产品，实际发货时按 BOM 展开为组件
- BOM 类型选择「套件」

#### 拆解

- 将成品拆回原材料（如退货返工）
- 创建拆解单，生成组件入库和成品出库

---

### 4. 批次制造

#### 功能概览

连续生产模式下，一次投料生产多个批次。

#### 业务操作

- **合并 MO**：将多个小订单合并为一个大的生产批次
- **分批入库**：一个 MO 分多次完工入库
- **批次追溯**：记录每个批次的原料来源和生产参数

---

## 六、PLM 产品生命周期管理 (企业版)

### 1. ECO 工程变更单

#### 字段解释

| 字段 | 说明 |
|------|------|
| **ECO 编号** | 变更单唯一标识 |
| **类型** | BOM 变更 / 工艺变更 / 文档变更 |
| **影响产品** | 受变更影响的产品范围 |
| **变更原因** | 成本优化 / 质量问题 / 法规要求等 |
| **生效日期** | 变更正式生效时间 |
| **审批状态** | 草稿 → 审批中 → 已批准 → 已实施 |

#### 业务流程

```
变更申请 → 影响评估 → 审批 → 试产验证 → 生效 → 通知相关方
```

#### 业务操作

- **创建 ECO**：PLM → 工程变更 → 新建
- **BOM 对比**：查看变更前后的 BOM 差异
- **审批流程**：多级审批后生效
- **版本升级**：变更后 BOM 版本号自动升级

---

### 2. 版本管理

#### 业务操作

- **BOM 版本**：查看产品历史 BOM 版本
- **生效管理**：设置新版本生效日期，旧版本自动失效
- **版本追溯**：查看某产品用过哪些版本的 BOM

## 七、设备维护 (企业版)

### 1. 预防性维护

#### 字段解释

| 字段 | 说明 |
|------|------|
| **设备** | 需要维护的生产设备 |
| **维护类型** | 保养 / 校准 / 巡检 |
| **频率** | 按时间（每月）或按用量（每 1000 小时） |
| **维护团队** | 负责维护的工程师团队 |
| **检查清单** | 维护作业的检查项 |

#### 配置步骤

1. **创建设备档案**
   - 维护 → 设备 → 新建
   - 关联工作中心、供应商（维保商）

2. **设置维护计划**
   - 创建预防性维护规则
   - 设置频率和提醒提前期

#### 业务操作

- **维护日历**：查看月度维护计划
- **到期提醒**：提前通知维护团队准备
- **维护记录**：记录每次维护的内容、更换的备件、工时
- **MTBF/MTTR**：统计平均故障间隔和平均修复时间

---

### 2. 故障维修

#### 业务流程

```
故障报警 → 创建维修单 → 停机记录 → 维修执行 → 恢复运行 → 根因分析
```

#### 业务操作

- **快速报修**：车间人员扫描设备条码报修
- **维修派工**：分配给维修工程师
- **备件领用**：从仓库领用维修备件
- **停机统计**：记录停机时长，影响 OEE 计算
- **根因分析**：5 Why 分析，防止复发

## 附录

### 制造流程检查清单

#### 生产前

- [ ] BOM 已审核且为最新版本
- [ ] 原材料库存充足或已安排采购
- [ ] 工作中心可用，无维护计划冲突
- [ ] 工艺路线已配置正确

#### 生产中

- [ ] 按工序顺序执行，及时报工
- [ ] 异常情况（报废、返工）及时记录
- [ ] 物料损耗在标准范围内

#### 生产后

- [ ] 完工数量与计划一致（或在容差内）
- [ ] 质检合格，已入库
- [ ] 实际成本已归集，差异分析完成

### 生产 KPI 参考

| 指标 | 健康值 | 说明 |
|------|--------|------|
| 计划达成率 | > 95% | 按时完成订单占比 |
| OEE | > 85% | 设备综合效率 |
| 一次合格率 | > 98% | 无需返工的比例 |
| 材料损耗率 | < 3% | 实际用量 vs BOM 用量 |
| 生产周期 | 行业相关 | 下单到完工的平均天数 |
