## 一、POS 基础配置

### 1. POS 会话设置

#### 字段解释

| 字段 | 说明 |
|------|------|
| **POS 名称** | 如「门店一收银台」 |
| **销售日记账** | 关联的会计日记账 |
| **发票日记账** | 开发票时使用的日记账 |
| **默认客户** | 未选择客户时的默认客户 |
| **价格表** | 适用的产品价格表 |
| ** warehouse ** | 出货仓库 |

#### 配置步骤

- POS → 配置 → POS → 新建
- 设置关联的日记账、仓库、价格表

---

### 2. 产品配置

#### 业务操作

- **上架产品**：在产品中勾选「可在 POS 销售」
- **条码**：录入产品条码，支持扫码枪扫描
- **称重产品**：设置按重量计价，连接电子秤

---

### 3. 支付方式

#### 支持方式

| 方式 | 说明 |
|------|------|
| **现金** | 现金收款，计算找零 |
| **银行卡** | 刷卡支付 |
| **移动支付** | 微信/支付宝扫码 |
| **信用卡终端** | Stripe、Adyen 终端 |
| **会员余额** | 使用会员储值余额 |
| **积分抵扣** | 使用积分抵扣部分金额 |

---

### 4. 小票与打印

#### 配置步骤

- POS → 配置 → 小票模板
- 设置小票抬头、页脚、 Logo
- 配置 ESC/POS 打印机

---

## 二、零售收银

### 1. 开单

#### 业务操作

- **打开会话**：收银员登录，开启收银会话
- **添加产品**：扫描条码 / 点击产品图标 / 搜索
- **修改数量**：点击数量字段修改
- **选择客户**：搜索并选择会员客户

---

### 2. 收款

#### 业务操作

- **选择支付方式**：现金/银行卡/移动支付
- **现金收款**：输入实收金额，系统自动计算找零
- **多方式支付**：一笔订单使用多种支付方式
- **挂单**：临时保存订单，处理其他客户后恢复

---

### 3. 折扣与优惠

#### 业务操作

- **整单折扣**：按百分比或固定金额打折
- **单品折扣**：对某个商品打折
- **促销码**：输入优惠码享受折扣
- **会员价**：会员自动享受会员价格

---

### 4. 退货

#### 业务操作

- **查找原单**：输入原订单号或扫描小票条码
- **选择退货商品**：勾选要退的商品
- **退款方式**：原路退回 / 退现金 / 退到会员余额
- **退货小票**：打印退货凭证

---

### 5. 关班结算

#### 业务操作

- **关班**：结束当前收银会话
- **现金盘点**：清点现金，与系统记录对比
- **差异处理**：记录现金差异原因
- **交班报表**：打印当班销售汇总

## 三、餐厅 POS

### 1. 桌台管理

#### 业务操作

- **桌台布局**：在 POS 上可视化布置桌台位置
- **开台**：选择桌台，输入人数
- **转台**：将订单转移到另一桌
- **并台**：两桌合并结账

---

### 2. 点餐流程

#### 业务操作

- **点餐**：浏览菜单，点击菜品添加到订单
- **口味备注**：为菜品添加特殊要求（如少辣、免葱）
- **分单**：将不同菜品分到不同打印机（如前厅/后厨/酒水）
- **催菜**：标记某道菜加急

---

### 3. 厨房打印

#### 配置步骤

- 为不同菜品类别指定打印到不同厨房打印机
- 配置打印模板（菜品、备注、桌号、时间）

---

### 4. 分单与合单

#### 业务操作

- **AA 制分单**：一桌客人按人数或按菜品分拆账单
- **合单**：相邻桌合并结账

---

## 四、会员与积分

### 1. 会员注册

#### 业务操作

- **快速注册**：收银时输入手机号快速建档
- **填写信息**：补充姓名、生日等
- **实体卡**：发放实体会员卡，绑定会员号

---

### 2. 积分规则

#### 配置步骤

- POS → 配置 → 积分规则
- 设置消费金额与积分比例（如 1 元 = 1 积分）
- 设置积分有效期

---

### 3. 积分兑换

#### 业务操作

- **积分抵现**：结账时选择使用积分抵扣
- **兑换礼品**：在门店兑换积分礼品
- **积分查询**：收银台或会员自助查询余额

---

## 五、IoT 设备集成

### 1. IoT Box

#### 功能概览

IoT Box 是 Odoo 的物联网硬件盒子，连接各种外设到 POS 系统。

---

### 2. 硬件连接

#### 支持设备

| 设备 | 说明 |
|------|------|
| ** receipt printer ** | 小票打印机（ESC/POS） |
| **条码扫描枪** | USB/蓝牙扫码 |
| **电子秤** | 称重产品自动计价 |
| **钱箱** | 现金抽屉 |
| **客户显示屏** | 面向客户的金额显示 |
| **刷卡终端** | Stripe/Adyen 读卡器 |
| **门禁** | 与会员系统联动 |

#### 配置步骤

- 将 IoT Box 接入局域网
- 在 Odoo 中配置 IoT Box IP 地址
- 连接外设，系统自动识别
