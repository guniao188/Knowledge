## 一、员工管理

### 1. 员工档案

#### 字段解释

| 字段 | 说明 |
|------|------|
| **姓名** | 员工姓名 |
| **工号** | 公司内部编号 |
| **部门** | 所属部门 |
| **职位** | 岗位名称 |
| **经理** | 直属上级 |
| **入职日期** | 正式入职日期 |
| **工作状态** | 在职 / 离职 / 试用期 / 实习 |
| **工作邮箱** | 企业邮箱 |
| **私人联系方式** | 电话、紧急联系人 |
| **银行账号** | 工资发放账户 |
| **证件信息** | 身份证号、护照号等 |
| **学历** | 教育背景 |
| **工作经历** | 过往工作履历 |

#### 业务操作

- **创建员工**：HR → 员工 → 新建
- **上传附件**：合同扫描件、学历证书、资格证书
- **试用期管理**：设置试用期到期提醒
- **转正流程**：试用期结束后发起转正评估
- **离职流程**：设置离职日期 → 触发交接清单 → 停用账户 → 结算工资

---

### 2. 组织架构

#### 业务操作

- **部门管理**：HR → 配置 → 部门 → 维护部门树
- **汇报关系图**：一键生成可视化组织架构图
- **编制管理**：为每个部门/岗位设置人员编制上限
- **成本中心**：部门关联会计科目，用于成本归集

---

### 3. 合同管理

#### 字段解释

| 字段 | 说明 |
|------|------|
| **合同类型** | 全职 / 兼职 / 实习 / 劳务 / 外包 |
| **合同期限** | 固定期限 / 无固定期限 |
| **起止日期** | 合同生效和到期日期 |
| **试用期** | 试用起止日期 |
| **薪资** | 合同约定薪资 |
| **工作时间** | 标准工时 / 综合工时 / 不定时 |
| **竞业限制** | 是否签订竞业协议 |

#### 业务操作

- **合同到期提醒**：提前 30/60/90 天自动提醒 HR 和经理
- **续签管理**：到期前发起续签流程
- **合同模板**：使用标准模板快速生成合同

---

### 4. 技能与证书

#### 业务操作

- **技能矩阵**：为员工添加技能标签和熟练度等级
- **证书管理**：记录证书名称、编号、发证机构、有效期
- **到期提醒**：证书到期前自动提醒更新
- **培训记录**：关联参加的培训课程和成绩

## 二、考勤管理

### 1. 打卡方式

| 方式 | 说明 |
|------|------|
| **网页打卡** | 通过浏览器访问 Odoo 打卡 |
| **手机 APP** | GPS 定位打卡 |
| **Kiosk 终端** | 前台公共设备，PIN 码或人脸识别 |
| **门禁集成** | 与第三方门禁系统对接 |
| **手动补卡** | 异常情况由经理或 HR 补录 |

#### 配置步骤

- 考勤 → 配置 → 设置：启用需要的打卡方式
- 配置 → Kiosk：设置 Kiosk 模式和 PIN 码

---

### 2. 班次设置

#### 字段解释

| 字段 | 说明 |
|------|------|
| **班次名称** | 如「早班」「晚班」「弹性班」 |
| **上班时间** | 标准签到时间 |
| **下班时间** | 标准签退时间 |
| **弹性区间** | 允许提前/延后打卡的范围 |
| **休息时段** | 午休时间 |
| **加班规则** | 超过多久算加班 |

#### 配置步骤

- 考勤 → 配置 → 班次 → 新建
- 为员工分配默认班次

---

### 3. 考勤异常处理

#### 异常类型

| 类型 | 说明 |
|------|------|
| **迟到** | 晚于上班时间打卡 |
| **早退** | 早于下班时间打卡 |
| **缺勤** | 无打卡记录且无请假 |
| **漏打卡** | 只打了一次卡 |
| **外勤** | 因公外出未在公司打卡 |

#### 业务操作

- **异常报表**：考勤 → 报告 → 异常考勤
- **补卡申请**：员工提交补卡说明 → 经理审批
- **外勤登记**：提前申请外勤，记录外出事由
- **月度汇总**：自动生成月度考勤汇总表

---

## 三、请假管理

### 1. 假期类型

| 类型 | 说明 |
|------|------|
| **年假** | 按工龄分配的带薪假期 |
| **病假** | 因病请假，可能需要医院证明 |
| **事假** | 无薪事假 |
| **婚假** | 法定婚假 |
| **产假/陪产假** | 生育相关假期 |
| **丧假** | 直系亲属去世 |
| **调休** | 加班调休 |
| **公假** | 法定假日 |

---

### 2. 请假申请与审批

#### 业务操作

- **提交申请**：员工选择假期类型 → 选择日期 → 填写事由 → 提交
- **附件上传**：病假上传医院证明
- **审批流程**：直属经理 → 部门总监 → HR（按假期类型配置）
- **审批通知**：审批人收到待办通知
- **日历同步**：批准后自动同步到日历

---

### 3. 假期余额

#### 配置步骤

- 请假 → 配置 → 假期类型 → 设置分配规则
- 按入职日期/工龄自动生成年假额度

#### 业务操作

- **余额查询**：员工随时查看各类假期剩余天数
- **结转规则**：设置年假是否可延期至下一年
- **清零提醒**：年底提醒员工使用剩余年假

---

## 四、工时表

### 1. 工时填报

#### 填报方式

| 方式 | 说明 |
|------|------|
| **网格视图** | 一行一个项目，按天填写小时数 |
| **日历视图** | 在日历上拖拽时间段 |
| **任务内填报** | 在项目任务上直接记录工时 |
| **计时器** | 开始工作时点击计时，结束自动记录 |

#### 字段解释

| 字段 | 说明 |
|------|------|
| **日期** | 工作日期 |
| **项目** | 归属项目 |
| **任务** | 具体任务 |
| **描述** | 工作内容 |
| **工时** | 投入小时数 |
| **类型** | 正常 / 加班 / 出差 |

---

### 2. 审批流程

#### 业务操作

- **提交工时表**：员工按周/月提交工时表
- **项目经理审批**：确认工时合理性和准确性
- **驳回修改**：退回给员工补充或修正
- **锁定**：审批通过后锁定，不可修改

---

### 3. 工时分析

#### 报表类型

- 个人工时报表（周/月/年）
- 项目工时汇总
- 部门工时统计
- 工时利用率分析

## 五、招聘管理

### 1. 职位发布

#### 字段解释

| 字段 | 说明 |
|------|------|
| **职位名称** | 招聘岗位 |
| **部门** | 所属部门 |
| **招聘人数** | 需求人数 |
| **工作地点** | 办公地点 |
| **薪资范围** | 预算薪资区间 |
| **职位描述** | 岗位职责和要求 |
| **发布渠道** | 官网 / 招聘网站 / 内推 |

#### 业务操作

- **创建职位**：招聘 → 职位 → 新建
- **发布到网站**：开启后自动同步到招聘页面
- **招聘渠道追踪**：记录候选人来源，分析渠道效果

---

### 2. 简历管理

#### 业务操作

- **简历收取**：
  - 邮箱自动解析：发送简历到指定邮箱自动生成候选人
  - 手动录入
  - 批量导入
- **简历筛选**：标记通过/不通过，填写筛选意见
- **人才库**：未录用但优秀的候选人归档到人才库

---

### 3. 面试安排

#### 业务操作

- **安排面试**：选择面试官 → 系统自动检查日历空闲 → 发送邀请
- **面试评价表**：面试官填写结构化评价
- **多轮面试**：记录每轮面试的结果和反馈
- **背调**：标记需要背景调查

---

### 4. Offer 发放

#### 业务操作

- **生成 Offer**：通过后自动生成 Offer 信
- **电子签章**：发送 Offer 给候选人电子签署
- **入职准备**：确认接受后，触发入职流程
  - 创建员工档案
  - 准备工位和设备
  - 安排入职培训

## 六、绩效评估

### 1. 评估计划

#### 字段解释

| 字段 | 说明 |
|------|------|
| **评估周期** | 年度 / 半年度 / 季度 |
| **评估模板** | KPI / 行为指标 / 360°反馈 |
| **评估人** | 自评 / 经理评 / 同事评 / 下属评 |
| **权重** | 各评估维度的权重比例 |

#### 配置步骤

- 评估 → 配置 → 评估计划 → 新建
- 设置启动日期、自动提醒

---

### 2. 360° 评估

#### 业务操作

- **多维度打分**：员工自评 + 经理评价 + 同事互评 + 下属评价
- **匿名评价**：同事/下属评价可设为匿名
- **校准会议**：经理层横向校准评分，消除偏差
- **绩效面谈**：一对一反馈评估结果

---

### 3. 目标管理 (OKR)

#### 业务操作

- **设定目标**：年初/季度初设定个人 OKR
- **关键结果**：每个目标下设 2-4 个可量化的关键结果
- **进度更新**：定期更新关键结果的完成百分比
- **复盘**：周期结束复盘目标达成情况

---

## 七、薪资管理 (企业版)

### 1. 薪资结构

#### 字段解释

| 字段 | 说明 |
|------|------|
| **基本工资** | 固定月薪/年薪 |
| **岗位津贴** | 岗位补贴 |
| **绩效奖金** | 与绩效挂钩的浮动部分 |
| **加班费** | 法定加班工资 |
| **社保公积金** | 个人和公司缴纳部分 |
| **个税** | 代扣代缴个人所得税 |
| **实发工资** | 最终到手金额 |

#### 配置步骤

- 薪资 → 配置 → 薪资结构 → 新建工资项
- 设置每个工资项的类型（收入/扣除/公司成本）
- 配置计算公式（Python 表达式）

---

### 2. 工资单生成

#### 业务操作

- **批量生成**：按周期（月/双周/周）批量生成工资单
- **自动计算**：根据考勤、工时、绩效自动计算应发/应扣
- **异常检查**：检查为负工资、异常高额等
- **审批**：HR 审核 → 财务审核 → 总经理审批（大额）
- **发放**：导出银行代发文件 → 发放工资

---

### 3. 社保与个税

#### 配置步骤

- 安装本地化薪资模块（如中国版）
- 配置社保基数、比例、公积金比例
- 配置个税累进税率表（自动更新）

#### 业务操作

- **社保计算**：按当地政策自动计算个人和公司缴纳额
- **个税计算**：累计预扣法自动计算个税
- **申报导出**：导出社保和个税申报表

### 2. 午餐管理

#### 业务操作

- **在线点餐**：员工在 APP 上选择当日餐品
- **汇总下单**：管理员按餐厅汇总订单

#### 业务操作

- **分享职位**：员工分享招聘链接到社交网络
- **提交推荐**：填写被推荐人信息
- **进度跟踪**：查看推荐状态（初筛/面试/Offer/入职）

---

- **访客预约**：外部人员在线预约访问
- **现场签到**：访客扫码或前台代为登记
- **通知被访人**：自动通知员工有访客到达
- **安全合规**：打印访客证、记录紧急联系人
