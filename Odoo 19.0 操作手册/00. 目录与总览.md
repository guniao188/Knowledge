## 阅读指南

### 标题层级约定

| 层级 | 含义 |
|------|------|
| `#` | 文件主标题（对应一个大类） |
| `##` | 子系统/应用名称 |
| `###` | 具体功能模块 |
| `####` | 字段解释 |
| `#####` | 配置步骤 / 业务操作 |

### 标记约定

- > **待补充**：内容框架已建立，需根据实际业务场景补充细节
- > **注意**：操作前必须确认的事项
- > **警告**：可能导致数据丢失或系统异常的高风险操作

### 权限标识

文中涉及的操作按角色区分：
- **系统管理员**：需 Settings 权限
- **应用管理员**：需对应应用的 Configuration 权限
- **普通用户**：基础业务操作权限
- **访客**：门户/网站公开访问
