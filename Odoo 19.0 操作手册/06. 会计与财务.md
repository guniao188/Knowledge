## 一、基础配置

### 1. 会计科目表

#### 字段解释

| 字段 | 说明 |
|------|------|
| **科目代码** | 唯一编码，通常按层级结构（如 1000 资产类） |
| **科目名称** | 科目描述 |
| **科目类型** | 资产 / 负债 / 权益 / 收入 / 费用 / 成本等 |
| **内部类型** | 应收 / 应付 / 银行 / 现金 /  view 等 |
| **允许对账** | 是否启用该科目的往来核销功能 |
| **币种** | 限制该科目仅记录特定币种（如外币银行） |
| **集团科目** | 合并报表时的映射科目 |

#### 配置步骤

1. **加载会计科目表**
   - 安装时选择国家本地化包自动加载
   - 或会计 → 配置 → 会计科目表 → 手动导入

2. **维护二级科目**
   - 会计 → 配置 → 会计科目 → 新建
   - 根据前缀自动归类（如 2201 为应付账款子科目）

3. **科目组**
   - 会计 → 配置 → 科目组
   - 用于报表层级汇总

---

### 2. 日记账

#### 字段解释

| 字段 | 说明 |
|------|------|
| **日记账名称** | 如「销售发票」「银行」「现金」 |
| **类型** | 销售 / 采购 / 银行 / 现金 / 杂项 |
| **默认借方科目** | 该日记账的默认借方科目 |
| **默认贷方科目** | 该日记账的默认贷方科目 |
| **编号序列** | 凭证号的自动生成规则 |
| **分录过账** | 自动 / 手动过账 |

#### 配置步骤

- 会计 → 配置 → 日记账 → 新建
- 为每个银行账户、现金账户、发票类型建立独立日记账

---

### 3. 会计期间与财年

#### 配置步骤

- 会计 → 配置 → 设置 → 财年
- 设置财年起止日期（如 1月1日-12月31日，或4月1日-3月31日）
- 系统自动生成会计期间

---

### 4. 币种与汇率

参见 [[01. 系统基础与配置]] → 币种与汇率

#### 会计特有配置

- 会计 → 配置 → 自动汇率 → 设置自动更新频率
- 汇率差异科目：设置汇兑损益的入账科目

---

### 5. 税种配置

#### 字段解释

| 字段 | 说明 |
|------|------|
| **税种名称** | 如「13% 增值税（销项）」 |
| **税率** | 百分比 |
| **税类型** | 销售税 / 采购税 |
| **科目映射** | 税金对应的会计科目（应交税费-销项税） |
| **含税/不含税** | 价格是否包含税金 |
| **适用范围** | 国家 / 地区 |

#### 配置步骤

- 会计 → 配置 → 税种 → 新建
- 设置税率、类型、科目映射
- 在产品/服务上绑定默认税种

## 二、应收管理

### 1. 客户发票

#### 创建方式

| 方式 | 说明 |
|------|------|
| **从销售订单** | 销售模块确认订单后一键开票 |
| **从交货单** | 按实际发货数量开票 |
| **手工发票** | 会计直接创建 |
| **定期开票** | 订阅/服务合同的周期性开票 |

#### 字段解释

| 字段 | 说明 |
|------|------|
| **客户** | 开票对象 |
| **发票日期** | 发票开具日期 |
| **到期日** | 付款截止日期 |
| **发票行** | 产品/服务、数量、单价、税、小计 |
| **付款条款** | 如「30天」「月结」 |
| **过账状态** | 草稿 → 已发送 → 已支付 |

#### 业务操作

- **发送发票**：点击「发送邮件」，客户收到 PDF 发票
- **打印发票**：生成正式发票 PDF
- **发票审批**：如需审批流，确认后过账
- **寄送状态**：标记发票已邮寄

---

### 2. 付款登记与核销

#### 业务操作

- **登记收款**：会计 → 客户 → 付款 → 新建
  - 选择客户、金额、付款方式（银行/现金）
  - 系统自动匹配未结发票

- **核销**：
  - 自动核销：系统按到期日自动匹配发票
  - 手动核销：选择具体要核销的发票
  - 部分核销：付款金额小于发票金额

- **银行手续费**：登记时单独记录手续费金额和科目

---

### 3. 催款与对账

#### 催款流程

1. **查看逾期**：会计 → 客户 → 逾期发票
2. **发送催款信**：
   - 一级：友好提醒（到期后 5 天）
   - 二级：正式催款（到期后 15 天）
   - 三级：最后通牒（到期后 30 天）
3. **电话跟进**：记录沟通结果
4. **暂停供货**：通知销售暂停该客户新订单
5. **法务追讨**：委托律师或走法律程序

#### 对账

- 定期向客户发送对账单（Statement）
- 客户确认后回签

---

### 4. 贷项通知单（红字发票）

#### 使用场景

- 销售退回
- 价格折让
- 发票错误冲销

#### 业务操作

- 在原发票上点击「添加贷项通知单」
- 选择原因：退款 / 折扣 / 错误
- 系统自动生成红字凭证，冲减应收

---

## 三、应付管理

### 1. 供应商账单

参见 [[03. 采购与供应链]] → 供应商账单

#### 补充字段

| 字段 | 说明 |
|------|------|
| **到期日** | 基于付款条款自动计算 |
| **已付金额** | 累计已付款 |
| **剩余金额** | 待付金额 |
| **挂账状态** | 正常 / 争议 / 待审批 |

---

### 2. 付款

#### 业务操作

- **付款单**：会计 → 供应商 → 付款 → 新建
- **批量付款**：勾选多张账单批量付款
- **付款方式**：银行转账 / 支票 / 现金 / 在线支付
- **付款审批**：大额付款需多级审批

---

### 3. 预付款

#### 业务操作

- **客户预收款**：提前收到客户款项，后续抵扣发票
- **供应商预付款**：提前支付给供应商，后续抵扣账单
- **预付款核销**：在后续发票/账单上指定使用预付款

---

## 四、费用报销

### 1. 费用提交

#### 字段解释

| 字段 | 说明 |
|------|------|
| **员工** | 报销人 |
| **费用类别** | 差旅 / 餐饮 / 办公 / 通讯等 |
| **日期** | 费用发生日期 |
| **金额** | 报销金额 |
| **币种** | 如外币需按汇率折算 |
| **收据** | 发票/小票照片附件 |
| **说明** | 费用事由 |
| **客户/项目** | 可关联到具体客户或项目 |

#### 业务操作

- **提交报销**：员工在「费用」应用填写并提交
- **拍照上传**：手机 APP 拍照上传收据
- **多币种**：外币费用自动按汇率折算为本位币
- **里程报销**：输入里程数，按标准费率自动计算金额

---

### 2. 审批与过账

#### 审批流程

```
员工提交 → 直属经理审批 → 财务审核 → 过账 → 付款
```

#### 业务操作

- **批量审批**：经理在列表视图批量批准/拒绝
- **退回修改**：审批人可要求补充材料或修改
- **过账**：财务确认后生成会计凭证
- **付款**：过账后进入付款流程（可合并到工资发放）

---

### 3. 费用报表

#### 报表类型

- 按员工汇总月度费用
- 按类别分析费用结构
- 按项目/客户归集费用
- 预算 vs 实际对比

## 五、银行与现金

### 1. 银行对账

#### 功能概览

将系统内的银行交易记录与实际银行对账单进行匹配，确保账实相符。

#### 业务操作

- **开始对账**：会计 → 银行 → 对账 → 选择账户和月份
- **匹配交易**：
  - 自动匹配：系统按金额和日期自动建议匹配
  - 手动匹配：选择系统交易和银行交易配对
  - 多对一匹配：多笔小额合并匹配一笔大额
- **未达账项**：已付/已收但银行未入账的项目
- **银行手续费/利息**：直接在对账界面录入

#### 对账完成

- 差异为零时确认对账完成
- 系统生成对账报告

---

### 2. 银行对账单导入

#### 支持格式

| 格式 | 说明 |
|------|------|
| **OFX** | 标准银行交换格式 |
| **QIF** | Quicken 交换格式 |
| **CSV** | 通用表格格式（需映射列） |
| **MT940** | SWIFT 银行对账报文 |
| **CAMT.053** | ISO 20022 格式 |

#### 业务操作

- 会计 → 银行 → 导入对账单 → 上传文件
- 系统自动识别交易并建议匹配
- 处理无法自动匹配的交易

---

### 3. 内部转账

#### 业务操作

- 会计 → 会计分录 → 转账
- 从一个银行/现金账户转到另一个
- 系统自动生成借贷双方的凭证

## 六、固定资产

### 1. 资产卡片

#### 字段解释

| 字段 | 说明 |
|------|------|
| **资产名称** | 固定资产描述 |
| **资产类别** | 房屋 / 设备 / 车辆 / 电子设备等 |
| **购置日期** | 入账日期 |
| **原值** | 购置成本 |
| **残值** | 预计报废时的剩余价值 |
| **使用年限** | 预计使用月数 |
| **折旧方法** | 直线法 / 双倍余额递减法 / 年数总和法等 |
| **当前净值** | 原值 - 累计折旧 |

#### 配置步骤

- 会计 → 固定资产 → 资产 → 新建
- 填写基本信息、成本信息、折旧参数
- 确认后系统自动生成折旧计划表

---

### 2. 折旧方法

| 方法 | 说明 |
|------|------|
| **直线法** | 每年折旧 = (原值 - 残值) / 使用年限 |
| **双倍余额递减法** | 加速折旧，早期折旧额高 |
| **年数总和法** | 加速折旧，按剩余年限比例计算 |
| **工作量法** | 按实际使用工时/里程计算 |

#### 折旧运行

- 会计 → 固定资产 → 运行折旧
- 按月/季度/年度运行，自动生成折旧凭证

---

### 3. 资产处置

#### 业务操作

- **出售**：录入出售价，计算处置损益
- **报废**：残值为零，确认报废损失
- **转让**：资产转移到其他公司或部门
- **重估**：按公允价值重新评估资产价值

## 七、税务

### 1. 销项税

#### 业务操作

- 销售发票自动计算销项税
- 按税率汇总销项税额
- 查看应交增值税明细

---

### 2. 进项税

#### 业务操作

- 供应商账单自动计算进项税
- 进项税抵扣管理
- 不可抵扣进项税的处理（如用于免税项目）

---

### 3. 纳税申报

#### 配置步骤

- 安装对应国家的本地化税务报表模块
- 配置纳税申报表模板

#### 业务操作

- **税务报表**：会计 → 报表 → 纳税申报
- 自动生成增值税申报表
- 进项/销项/应缴税额自动汇总
- 导出税务系统要求的格式

## 八、期末结账

### 1. 期末调整

#### 调整分录类型

| 类型 | 说明 |
|------|------|
| **计提折旧** | 固定资产折旧 |
| **计提工资** | 已发生未支付的工资 |
| **预提费用** | 已发生未支付的其他费用 |
| **待摊费用** | 预付费用的本期摊销 |
| **坏账准备** | 应收账款坏账计提 |
| **存货跌价** | 库存减值准备 |

#### 业务操作

- 会计 → 会计分录 → 日记账分录 → 新建
- 手工录入调整分录
- 或使用「期末结转」自动化规则

---

### 2. 汇兑损益

#### 业务操作

- 会计 → 会计分录 → 汇兑损益
- 选择币种和日期
- 系统自动计算外币账户的汇率差异
- 生成汇兑损益调整分录

---

### 3. 关账与锁定期

#### 字段解释

| 字段 | 说明 |
|------|------|
| **非顾问锁定日期** | 普通用户不能修改此日期之前的凭证 |
| **所有用户锁定日期** | 包括顾问也不能修改 |

#### 业务操作

- 会计 → 会计 → 锁定日期
- 设置锁定期，防止已结账期间的凭证被修改
- 如需调整，需先解除锁定（留下审计痕迹）

---

## 九、财务报表

### 1. 资产负债表

#### 报表结构

| 区域 | 内容 |
|------|------|
| **资产** | 流动资产 + 非流动资产 |
| **负债** | 流动负债 + 非流动负债 |
| **所有者权益** | 实收资本 + 留存收益 |

#### 业务操作

- 会计 → 报表 → 资产负债表
- 选择日期（通常为期末最后一天）
- 查看/导出/打印报表

---

### 2. 利润表

#### 报表结构

| 区域 | 内容 |
|------|------|
| **营业收入** | 主营业务收入 + 其他业务收入 |
| **营业成本** | 主营业务成本 |
| **毛利** | 营业收入 - 营业成本 |
| **期间费用** | 销售 / 管理 / 财务费用 |
| **营业利润** | 毛利 - 期间费用 |
| **净利润** | 扣除所得税后的利润 |

#### 业务操作

- 会计 → 报表 → 利润表
- 选择期间（如本月、本季度、本年度）
- 对比去年同期或预算数据

---

### 3. 现金流量表

#### 报表结构

| 区域 | 内容 |
|------|------|
| **经营活动** | 销售收款、采购付款、工资支付等 |
| **投资活动** | 购置/处置固定资产 |
| **筹资活动** | 借款、还款、分红 |

#### 配置步骤

- 会计科目 → 标签：为现金流量表科目打上标签
- 或直接法/间接法配置

---

### 4. 科目余额表

#### 业务操作

- 会计 → 报表 → 科目余额表
- 查看所有科目的期初、本期发生、期末余额
- 支持按期间、币种、公司筛选
- 点击金额可穿透查看明细凭证
