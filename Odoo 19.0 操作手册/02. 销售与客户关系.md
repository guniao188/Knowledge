## 一、CRM 客户关系管理

### 1. 销售线索管理

#### 功能概览

线索 (Lead) 是 CRM 漏斗的最前端，通常来自网站表单、邮件、展会、购买名单等渠道，经过资格审核后转化为商机。

#### 字段解释

| 字段 | 说明 |
|------|------|
| **线索标题** | 简要描述需求 |
| **公司名称** | 潜客所属公司 |
| **联系人** | 具体对接人 |
| **邮箱/电话** | 联系方式 |
| **来源** | 线索渠道（网站、邮件、展会、转介绍等） |
| **优先级** | 低 / 中 / 高 |
| **分配对象** | 负责跟进的销售员 |
| **标签** | 用于分类筛选 |
| **线索评分** | 自动化评分规则计算的值 |

#### 配置步骤

1. **线索来源配置**
   - CRM → 配置 → 设置：启用「线索」模式（默认商机模式可切换为线索+商机模式）
   - 配置 → 来源：维护获客渠道列表

2. **线索分配规则**
   - CRM → 配置 → 设置：启用「自动分配」
   - 按团队、地域、轮询等方式自动分配新线索

3. **评分规则**
   - CRM → 配置 → 评分规则：设置自动评分条件
   - 如「有邮箱 +10 分」「有电话 +5 分」「行业=IT +15 分」

#### 业务操作

- **创建线索**：手动新建 / 邮件自动创建 / 网站表单导入 / 批量导入 Excel
- **线索处理**：打电话 → 补充信息 → 判断意向 → 合格则「转化为商机」
- **批量分配**：勾选线索 → 行动 → 更改分配 → 选择销售员
- **丢失线索**：标记丢失并填写原因，用于后续分析
- **线索回收**：长时间未跟进的线索自动回收至公海池

### 2. 商机管理

#### 字段解释

| 字段 | 说明 |
|------|------|
| **商机名称** | 描述性标题 |
| **客户** | 关联的合作伙伴 |
| **预期收入** | 预估成交金额 |
| **成功率** | 赢单概率（如 30%、70%） |
| **加权收入** | 预期收入 × 成功率 |
| **预计成交日** | 预估的签约时间 |
| **阶段** | 销售漏斗阶段 |
| **下一步行动** | 待办事项 |
| **竞争对手** | 竞争对手信息 |

#### 配置步骤

1. **销售阶段**
   - CRM → 配置 → 阶段：按团队设置漏斗阶段
   - 如：初步接触 → 需求确认 → 方案报价 → 商务谈判 → 赢单/输单
   - 可设置阶段的「折叠」状态和「概率默认值」

2. **商机类型**
   - 新建商机 vs 重复购买（Upsell）
   - 启用「重复业务」追踪老客户的新需求

#### 业务操作

- **商机创建**：从线索转化 / 直接在客户卡片上新建
- **阶段推进**：在看板视图中拖拽卡片到下一阶段
- **收入预测**：基于各阶段商机和概率计算加权预测
- **赢单/输单**：点击按钮标记结果，输单时记录原因
- **活动安排**：在商机上安排电话、会议、邮件等跟进活动
- **报价关联**：在商机中直接创建报价单，成交后自动关联

#### 商机看板操作

| 操作 | 说明 |
|------|------|
| **+ 新建** | 在指定阶段快速创建商机 |
| **拖拽** | 变更商机阶段，可触发自动化规则 |
| **颜色标记** | 按标签/优先级显示不同颜色 |
| **快速编辑** | 点击卡片上的字段直接修改 |

### 3. 销售团队与目标

#### 配置步骤

1. **创建团队**
   - CRM → 配置 → 销售团队
   - 设置团队名称、团队经理、成员、邮箱别名

2. **设定目标**
   - 团队卡片 → 收入目标
   - 按月度/季度设置团队和个人收入目标

3. **仪表盘**
   - 为每个团队配置专属仪表盘
   - 显示漏斗分析、活动统计、收入达成率

#### 业务操作

- **团队切换**：顶部导航可切换关注的销售团队
- **业绩排行**：查看团队内销售员的业绩排名
- **目标追踪**：仪表盘显示当前完成进度 vs 目标

### 4. 活动与任务

#### 配置步骤

1. **活动类型**
   - CRM → 配置 → 活动类型：电话、会议、邮件、待办等
   - 设置默认建议、触发动作

2. **计划活动**
   - 在商机/客户卡片上点击「活动」→ 安排活动
   - 选择类型、日期、负责人、备注

#### 业务操作

- **活动日历**：在看板/日历视图查看待办活动
- **批量安排**：勾选多个记录批量安排跟进活动
- **活动完成**：完成后标记并记录结果
- **活动提醒**：到期前系统通知

|------|------|

| **销售金额** | 累计成交金额 |

### 6. 报表与分析

#### 报表类型

| 报表 | 说明 |
|------|------|
| **漏斗分析** | 各阶段转化率、瓶颈识别 |
| **线索来源分析** | 各渠道线索量、转化率、ROI |
| **销售活动分析** | 电话/会议/邮件量统计 |
| **收入预测** | 基于商机概率的加权预测 |
| **客户分析** | 成交客户、流失客户、复购率 |
| **流失原因分析** | 输单原因分布 |

#### 业务操作

- **透视表分析**：拖拽字段进行多维度分析
- **图表展示**：漏斗图、趋势图、对比图
- **导出报告**：Excel / PDF 导出
- **仪表盘**：将常用报表固定到个人仪表盘

## 二、销售管理

### 1. 报价单

#### 字段解释

| 字段 | 说明 |
|------|------|
| **客户** | 报价对象 |
| **报价单号** | 自动生成的编号 |
| **有效期** | 报价失效日期 |
| **价格表** | 应用的定价策略 |
| **付款条款** | 账期约定 |
| **交货条款** | Incoterm（如 FOB、CIF） |
| **订单行** | 产品、描述、数量、单价、折扣、小计 |
| **小计/税额/总计** | 自动计算 |

#### 配置步骤

1. **报价模板**
   - 销售 → 配置 → 报价模板
   - 预设常用产品组合、条款、页眉页脚
   - 可设置「推荐选项」（Upsell）

2. **产品配置器**
   - 配置产品的变体属性（颜色、尺寸等）
   - 设置变体间的加价/减价规则

#### 业务操作

- **创建报价**：销售 → 报价单 → 新建 / 从商机转化
- **添加产品**：直接选择 / 从配置器选择变体
- **编辑行项目**：修改数量、折扣、描述
- **可选产品**：推荐附加产品（如买电脑推荐鼠标）
- **发送报价**：点击「发送邮件」，客户收到带签章链接的报价
- **客户确认**：客户在线签字确认或点击接受
- **转化为订单**：确认后点击「确认订单」
- **报价修订**：已发送的报价可复制为新版并标注版本

---

### 2. 销售订单

#### 订单生命周期

```
报价单 → 确认订单 → 待发货 → 已发货 → 已开票 → 已收款
```

#### 业务操作

- **确认订单**：报价单确认后变为销售订单
- **查看库存**：订单行的「在手/预测」数量
- **预留**：为订单预留库存（取决于配置）
- **交货单**：自动或手动创建拣货单
- **分批交货**：一个订单分多次发货
- **开票策略**：
  - 按订单开票（确认即开票）
  - 按交货开票（发货后开票）
  - 按里程碑开票（项目型）

---

### 3. 产品配置器

#### 功能概览

用于产品变体较多时，通过向导引导客户选择属性组合。

#### 配置步骤

1. **启用变体**
   - 销售 → 配置 → 设置：启用「产品变体」

2. **定义属性**
   - 产品 → 配置 → 属性：创建属性（如颜色、尺寸、内存）
   - 设置属性值和对应的价格调整

3. **配置器展示**
   - 产品卡片 → 变体页签 → 自动生成变体矩阵
   - 可设置某些属性影响其他属性的可用值

#### 业务操作

- **前端配置**：客户在电商或门户上通过配置器选择变体
- **动态价格**：根据选择的属性组合实时计算价格
- **库存检查**：配置完成后检查对应变体的可用库存

---

### 4. 价格表与折扣

#### 价格表 (Pricelist)

| 类型 | 说明 |
|------|------|
| **公开价格表** | 所有客户可见（电商） |
| **分级价格表** | 不同客户等级不同价格（VIP、经销商） |
| **币种价格表** | 按币种定价 |
| **季节性价格表** | 特定时间段生效的价格 |

#### 价格规则

- **固定价格**：直接设定价格
- **折扣百分比**：在售价基础上打折
- **公式定价**：成本 +  markup / 售价 - 折扣
- **最小/最大保证金**：确保利润率

#### 配置步骤

1. **创建价格表**
   - 销售 → 产品 → 价格表
   - 设置币种、国家、客户分组

2. **添加规则**
   - 按产品 / 产品类别 / 所有产品
   - 设置最小数量（如批量折扣）
   - 设置生效日期范围

#### 业务操作

- **客户绑定**：在客户卡片上指定适用的价格表
- **订单选择**：报价时可切换价格表
- **价格刷新**：修改价格表后，未确认报价单可重新计算价格

---

### 5. 交货与开票策略

#### 交货策略

| 策略 | 说明 |
|------|------|
| **立即交货** | 有库存就发 |
| **按订单** | 等订单确认后统一拣货 |
| **按请求** | 手动触发发货 |

#### 开票策略

- **按订单开票**：订单确认即生成发票
- **按交货开票**：每次发货后生成发票
- **按固定比例**：如预付 30%，交货 70%
- **按里程碑**（项目型）：完成特定里程碑后开票

#### 配置步骤

- 销售 → 配置 → 设置 → 开票策略
- 或在客户卡片上单独设置默认策略

## 三、营销自动化

### 1. 营销工作流

#### 功能概览

基于触发条件自动执行营销活动，如发送邮件、创建任务、更新字段等。

#### 字段解释

| 字段 | 说明 |
|------|------|
| **工作流名称** | 营销活动标识 |
| **目标模型** | 作用于什么对象（联系人、商机、订单等） |
| **触发条件** | 何时启动（创建记录、字段变更、定时触发） |
| **过滤器** | 仅对满足条件的记录生效 |
| **活动节点** | 执行的动作（发邮件、等几天、更新字段） |

#### 配置步骤

1. **创建工作流**
   - 营销自动化 → 工作流 → 新建
   - 选择目标模型和触发条件

2. **设计流程**
   - 拖拽节点：发送邮件 / 等待 / 创建活动 / 更新字段
   - 设置节点间的等待时间
   - 配置条件分支（如「如果邮件已打开则...」）

3. **测试与启动**
   - 测试模式：仅影响测试记录
   - 启动后自动对新符合条件的记录执行

#### 业务场景

- **欢迎序列**：新注册用户自动发送 3 封欢迎邮件
- **弃购挽回**：购物车放弃 1 小时后发送优惠券
- **沉睡唤醒**：90 天未下单客户发送召回邮件
- **商机培育**：新线索自动发送产品介绍系列邮件

---

### 2. 邮件营销

#### 字段解释

| 字段 | 说明 |
|------|------|
| **邮件主题** | 收件人看到的标题 |
| **收件人列表** | 目标联系人分组 |
| **邮件正文** | HTML 内容，支持动态变量 |
| **发件人** | 显示的发件地址 |
| **发送时间** | 立即发送 / 定时发送 |
| **A/B 测试** | 对比不同主题/内容的效果 |

#### 配置步骤

1. **创建邮件列表**
   - 邮件营销 → 邮件列表：手动添加或从联系人筛选导入

2. **设计邮件**
   - 使用拖拽编辑器设计邮件模板
   - 插入动态内容块（如产品推荐、最新博客）

3. **发送与跟踪**
   - 发送后可查看打开率、点击率、退订率
   - 链接点击热力图分析

#### 业务操作

- **黑名单**：退订用户自动加入黑名单
- **重复排除**：同一活动避免向同一联系人重复发送
- **再营销**：向打开邮件但未点击的用户再次发送

---

### 3. 短信营销

#### 功能概览

通过 SMS 网关向联系人发送营销或通知短信。

#### 配置步骤

1. **配置 SMS 网关**
   - 设置 → 技术 → 短信：配置提供商（如 Twilio）
   - 或使用 Odoo IAP 积分

2. **创建短信活动**
   - 类似邮件营销，但内容更精简
   - 支持短链接跟踪点击

#### 业务场景

- **活动提醒**：活动开始前 1 天发送提醒
- **促销通知**：限时优惠即时推送
- **订单通知**：发货/签收短信通知

---

### 4. 社交媒体

#### 功能概览

Odoo 的社交营销模块（Social Marketing，企业版）可集中管理多个社媒账号。

#### 支持平台

- Facebook / Instagram
- Twitter / X
- LinkedIn
- YouTube（内容发布）

#### 业务操作

- **内容日历**：统一排期各平台发布计划
- **帖子创建**：富文本编辑 + 图片/视频
- **评论聚合**：在一个界面回复所有平台的评论
- **数据分析**：各平台互动量、粉丝增长统计

## 五、网站获客

### 1. 网站表单

#### 功能概览

在 Odoo 网站上创建表单，收集访客信息并自动创建 CRM 线索。

#### 配置步骤

1. **创建表单**
   - 网站 → 页面编辑 → 拖拽「表单」块到页面
   - 添加字段（姓名、邮箱、电话、需求描述）

2. **动作配置**
   - 提交后动作：创建线索 / 发送邮件 / 跳转到感谢页
   - 设置线索默认分配的销售团队

#### 业务操作

- **表单样式**：选择预设主题色和布局
- **必填验证**：标记必填字段
- **反垃圾**：启用 reCAPTCHA
- **跟踪来源**：通过 UTM 参数识别访客来源

---

### 2. 实时聊天

#### 功能概览

网站访客可实时与客服/销售对话，聊天记录自动关联 CRM。

#### 配置步骤

1. **创建频道**
   - 实时聊天 → 配置 → 频道
   - 设置名称、客服组、欢迎语

2. **网站集成**
   - 复制嵌入代码到网站
   - 或使用 Odoo 网站模块自动集成

3. **运营规则**
   - 在线/离开/离线状态自动切换
   - 无人时显示离线表单

#### 业务操作

- **客服分配**：按轮询 / 最少会话数 / 技能标签分配
- **快捷回复**：预设常见问题回复
- **聊天转商机**：对话中一键创建 CRM 线索
- **满意度评价**：会话结束后访客打分

---

### 3. 博客与内容营销

#### 功能概览

通过网站博客发布内容，吸引自然流量，培养潜在客户。

#### 配置步骤

1. **创建博客**
   - 网站 → 博客 → 新建文章
   - 使用富文本编辑器撰写内容
   - 设置封面图、标签、作者

2. **SEO 优化**
   - 设置页面标题、元描述、URL 别名
   - 关键词密度建议

3. **社交分享**
   - 配置分享按钮
   - 发布到社交媒体

#### 业务操作

- **评论管理**：审核访客评论
- **订阅**：访客订阅博客，新文章自动邮件通知
- **CTA**：在文章中插入「下载白皮书」等转化按钮

## 六、调查问卷

### 功能概览

创建表单、问卷、考试、评估，用于客户调研、员工满意度、招聘测评等。

#### 字段解释

| 字段 | 说明 |
|------|------|
| **问卷类型** | 问卷 / 投票 / 考试 |
| **问题类型** | 单选 / 多选 / 文本 / 矩阵 / 日期 / 上传文件 |
| **必填** | 是否必须回答 |
| **条件显示** | 根据前一题答案显示/隐藏 |
| **得分** | 考试模式下各题分值 |
| **证书** | 考试通过后自动生成证书 |

#### 配置步骤

1. **创建问卷**
   - 调查 → 新建 → 选择类型
   - 逐题添加，设置选项和条件

2. **样式设置**
   - 主题色、背景图、一页一题/全部展示

3. **发布与分享**
   - 生成公开链接
   - 嵌入网站
   - 邮件发送邀请

#### 业务场景

- **客户满意度**：订单完成后自动发送 NPS 调查
- **员工入职**：入职流程检查清单
- **招聘测评**：候选人在线完成能力测试
- **活动反馈**：活动结束后收集参会者评价
