# 迈田业务功能增强需求文档

## 1. 销售订单模块增强

### 1.1 销售订单新增字段

**预计工时**：5小时

| 字段名称 | 字段类型 | 描述 | 关联/自动生成 |
| --- | --- | --- | --- |
| 客户销售订单编号 | 文本 | 由客户名称+流水号自动生成 | 自动生成 |
| 运输方式 | 枚举 | 销售订单运输方式 | 枚举选项： - By Sea (海运) - By Train (火车) - By Air (空运) - By Truck (汽运) - By Express (快递) - Others (其他) 自动带到发货单、发票上 |
| 启运港 | 关联 | 销售订单的启运港口 | 关联港口表，自动带到发货单、发票上 |
| 启运港国家 | 关联 | 启运港所属国家 | 关联国家表，自动带到发货单、发票上 |
| 目的港 | 关联 | 销售订单的目的港口 | 关联港口表，自动带到发货单、发票上 |
| 目的港国家 | 关联 | 目的港所属国家 | 关联国家表，自动带到发货单、发票上 |
| 目的地 | 文本 | 目的地详细地址 | - |
| 收款账号 | 关联 | 公司银行账户 | 关联银行账户表（仅显示本公司账户），自动带到发票上 |
| 人民币订单总金额 | 货币 | 根据创建时汇率自动换算 | 自动计算 |
| 溢短装 | 关联 | 溢短装规则 | 关联溢短装表，自动带到发货单、发票上 |
| 总订单量 | 文本 | 默认值：1x40HQ | -，自动带到发货单、发票上 |
