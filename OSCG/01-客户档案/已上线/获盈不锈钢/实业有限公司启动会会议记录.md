# 获盈不锈钢实业有限公司启动会会议记录

**一、上线范围**

1. 目前第一个的上线公司抬头有几个？是否涉及多公司及内部公司间交易：1个抬头，不涉及内部交易
2. 部门架构和职能架构 ：10个操作人员+2个报工 ，其中电脑操作5人
3. 目前是否使用ERP软件，使用的ERP软件是什么：有使用系统，但现有系统功能有限，使用excel辅助管理
4. 预计上线时间

业务流程：产品入库--》根据客户订单去库存找对应尺寸的产品，若没有对应的尺寸则从加工方裁剪，其中原料放在加工方，加工后接收入库--》压扁，发货

**二、销售**

1. 销售具体业务流程？销售的单价是按件还是按吨 ：最终总价单位按照吨，在称重环节确定总价；报价和销售单的单位为件
2. 销售订单是否在Odoo中发送给客户确认？不需要
3. 各公司各团队的业务员有多少？销售4-5人，无须审批环节
4. 是否有价格表？销售员是否可以修改价格表？不需要
5. 销售订单价格是否含税？不含税
6. 销售订单如何开票，收款的流程和方式：有独立的财务系统，不需要在odoo开票。体现应收，不含税，需要管理成本

报表需求：当日、月度、订单的盈亏情况

1. 退货流程 ：退货不产生物流费用 ，退货的产品看能否返工，若不能放入库存

**三、采购**

1. 采购整体流程是否包含审批 没有，原料采购直接入加工厂，加工厂进行切割加工，

首先看自己仓库是否有库存，再考虑工厂加工，余料也会发到自己仓库

1. 计价方式是哪种，先进先出还是移动加权平均？先进先出，需要开启分析会计功能
2. 是否有完整的供应商档案，供应商档案是谁来维护？有
3. 退货流程 一般没有，忽略

**四、库存**

1. 目前有仓库？仓库和位置？其存储功能是什么？ ：1个，目前没有区分位置，成品当月或第二天会发货完成。余料有对应的编码，统一在同一个位置，
  残次品以位置区分

产品统计维度：编码、等级、厚度、宽度、进货时单价、重量

1. 是否有专人来管理仓库？1人
2. 仓库入库是否涉及质检？是，质检维度尺寸、重量
3. 现有库存是否精准？如不精准存在什么问题：精准
4. 仓库出库发货是否涉及审批流程？如有，请提供审批流 ：不涉及，销售发货单后需要制作通行证，交由门卫，打印销售单即可
5. 打包的费用是均摊到成本上还是直接进费用 ：做成灵活添加的形式

****

**五、制造**

1. 生产策略是MTO（Make-to-Order/按订单生产）还是MTS（Make-to-Stock/按库存生产）：MTO,根据订单进行自动化匹配库存及制造数量，等级-厚度-宽度 宽度可以通过拼接
2. BOM表是否完整？物料编码和半成品数量？BOM层级有几层？  没有固定的bom,有五种等级的不锈钢
3. 现有SKU(产品）数量及BOM数量
4. 委外加工类型是包工包料还是包工不包料 包工不包料
5. 拼卷裁切是否根据剩余宽度，拼卷的维度是什么
6. 以下环节是否符合贵司业务流程，或需要哪几个环节

接收入库：收货、质检、入库

制造：领料、制造、成品入库

发货：拣货、包装、发货

不需要统计设备上的数量，根据收发作业进行报工，一台机器一天10个单子左右

根据工序分配人员，每个工序大致2人，正手判断工序是否完成。由小组长进行确认工序是否完成

****

**六、会计**

1. 目前是否使用财务系统，会计模块目前是如何考虑，先上应收应付，还是全部？
2. 如果涉及多公司，账套如何管理？是否合并账套？内外帐是否一套？
3. 目前的业务涉及哪几个外币币种？是否需要多语言？

天然气、用电:按天平摊，每天费用不同

运输费：按单平摊，月结的方式

看财务报表时需要体现

重点：自动备料、自动排单
