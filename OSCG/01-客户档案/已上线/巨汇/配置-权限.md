# 权限

若修改代码后重启服务器后无效，可以看一下修改后的文件夹权限

若文件夹的权限不是odoo/odoo ，而是为odoo/root，则需要修改权限，直接修改最外面文件夹的权限

命令：

```
chown -R odoo:odoo /opt/odoo/odoo/odoo-server/addons
```
