# 自动备份功能

1、安装模块，模块位置为： addons目录: /mnt/odoo/odoo/custom/addons

并且addons平级配置Backup文件夹![image](_附件/1716278911512-8363e6ed-9a06-43e2-879a-bc-1a6c73c33e552e25.png)

2、若安装时出现缺少model的报错，则需要进入虚拟环境，下载模块

```
启用:
source /opt/odoo/odoo17/odoo17-venv/bin/activate
下载model：pip install XXX（报错的model）
pip install dropbox - i
pip install pyncclient
pip install boto3
pip install nextcloud-api-wrapper
退出:deactivate
```

修改文件夹权限： chown -R odoo:odoo /mnt/odoo

3、配置Backup Configuration![image](_附件/1716278949325-1aa92183-b8de-420a-9620-d2-64cb8011ae7f8553.png)

Database Name与当前数据库名称一致![image](_附件/1716279019254-bac55d28-91e4-4b70-a656-17-6ea2c8c36ed64dd1.png)
