# 单据编码取消自动编码，且重复时提醒

场景介绍：客户需要根据自己的编码规则对单据进行编号，取消系统的自动编码规则，但需要设置重复时提醒

思路：取消自动生成的单据编号和编号只读的代码逻辑，增加创建单据时的重复校验

STEP 1

创建订单时订单编号为"NEW"，点击保存时会生成编号。现需要取消自动编码，故将默认为NEW取消，设置为default为空，可将自动编码取消![image](_附件/1713248950736-d2ec0738-222e-4b96-9e62-31-e34213be177431f9.png)

![image](_附件/1713249589526-699dfc7c-6715-40aa-88fe-6e-744d928aa9d3ea9d.png)

STEP 2

增加代码限制重复的编号不允许创建

```
@api.constrains('name')
def check_name(selfs):
    for self in selfs:
        if selfs.search_count([
            ('name','=',self.name),
        ]) > 1:
            raise UserError(f'{self.name} 已存在')

@api.model
def create(cls, values):
    self = super().create(values)
    self.check_name()
    return self
```
