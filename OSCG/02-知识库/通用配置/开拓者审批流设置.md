修改密码重启命令 docker restart odoo

服务器动作

审批通过：

review = record.review_ids.filtered(lambda r:r.status=='pending')[:1]

reviewer_id = (review.reviewer_id| review.reviewer_ids)[:1].id

if reviewer_id:

record.with_user(reviewer_id).validate_tier()

审批拒绝：

review = record.review_ids.filtered(lambda r:r.status=='pending')[:1]

reviewer_id = (review.reviewer_id| review.reviewer_ids)[:1].id

if reviewer_id:

record.with_user(reviewer_id).reject_tier()

拒绝审批 record.with_context(dont_raise_write_error=1).write({'state':'draft'})

拒绝审批（同时把state改回草稿状态）：record.with_context(dont_raise_write_error=1).action_reset_approval_expense_sheets()

总经理审批 record.with_context(dont_raise_write_error=1).action_approve_expense_sheets()

财务过账审批 record.with_context(dont_raise_write_error=1).action_sheet_move_create()

部门主管审批 record.with_context(dont_raise_write_error=1).write({'state':'submit'})

批准、拒绝 隐藏action_approve_expense_sheets、action_refuse_expense_sheets

过账 隐藏action_sheet_move_create

登记付款 action_register_payment  state ！= 'post'
