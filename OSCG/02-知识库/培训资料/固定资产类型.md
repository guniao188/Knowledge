# 固定资产类型

- `[none]-无比例`
  ** (None - No proportion)**
- **用途**
  : 当选择此选项时，Odoo在计算折旧时不会对第一个或最后一个折旧期间的金额进行按天数比例的调整。这意味着无论资产是在期间的哪一天购置或处置，该期间的折旧金额都将是完整的期间折旧额（例如，一个月的完整折旧额），或者折旧会从下一个完整的期间开始。
- **影响**
  : 这种方法简化了计算，但可能不如按天数比例计算精确，尤其是在资产购置或处置日期不与折旧期间开始或结束日期完全吻合时。
- `[constant_periods]-恒定期问`
  ** (Constant periods)**
- **用途**
  : 这个选项通常与“直线折旧法”结合使用。它表示在资产的整个使用寿命内，每个完整的折旧期间（例如，每个月、每个季度或每年）的折旧金额是固定的、相等的。
- **影响**
  : 如果启用了“Prorata Temporis”功能（即按天数比例计算），那么第一个折旧期间的金额可能会根据实际天数进行调整，但随后的所有完整期间都将是恒定的折旧金额。如果未启用“Prorata Temporis”，则所有期间（包括第一个期间，如果折旧从购置当月开始）都将是恒定的金额。Odoo的直线折旧法就是将可折旧价值除以计划的折旧次数，使所有折旧分录的金额相同。
- `[daily_computation]-基于各期间`
  ** (Based on each period / Daily computation)**
- **用途**
  : 这个选项在Odoo中对应的是“Prorata Temporis”功能。它的目的是尽可能精确地计算资产的折旧。当启用此功能时，资产的第一个折旧分录将根据“按比例日期”（Prorata Date）和“首次折旧日期”（First Depreciation Date）之间剩余的时间来计算，而不是默认的完整期间金额。
- **影响**
  : 这意味着如果资产在折旧期间的中间日期被购置或处置，Odoo会根据资产在该期间内实际使用的天数来按比例计算折旧金额。例如，如果一个资产在一个30天的月份的第15天被购置，那么第一个月的折旧将只计算15天的金额。这使得折旧费用能够更准确地反映资产在特定会计期间内的实际消耗。
