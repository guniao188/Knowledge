# 生产部

1 核心业务流程

- 接收销售订单 → MRP运算 → 工单下达 → 领料 → 轧制/热处理/精整 → 入库

2 关键问题/痛点

- 是否有主生产计划（MPS）？是否考虑设备产能？
- 工艺路线是否复杂？是否需要系统帮助排产，如果需要，能否规定每道工序的标准工时
- 是否记录实际工时、能耗、废品率？
- 工人工资是计件还是固定工资 计件
- 一个产成品从加工到完成大概需要多长时间能完成
- 是否存在插单情况，插单是否频繁？插单的线下流程是怎样的
- 报废和拆解情况是否存在，频率大概多少，如果存在拆解，对应的业务场景是什么
- 报工是准备每个人报还是组长统一报

3 其他

- 现在部门协同工作中比较麻烦的事情是什么
- erp最需要解决什么问题
