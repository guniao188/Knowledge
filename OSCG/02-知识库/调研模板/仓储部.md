# 仓储部

1.核心业务流程

- 原材料入库 → 在库管理（分区/批次） → 生产领料 → 成品入库 → 销售出库

2.关键问题

1. 仓库是否区分室内仓、露天堆场等？不同区域管理方式是否不同？  光伏：一个人入库；另一个出库
2. 钢材成品（如卷板、棒材）如何标识？标签包含哪些信息：名称、厚度、长度、数量
3. 是否存在“同规格不同质”的情况？如何避免混放混发？两个产品
4. 发货时是否支持部分批次合并装车？系统/单据如何记录每件产品的原始批次？  有，需要回传收货签字单
5. 是否有地磅，称重是否集成？毛重-皮重=净重是否自动计算？是否与发货单关联？  点数
6. 是否有“在途库存”管理？（如已发货未签收、在港未清关）如何监控？  只有内销
7. 盘点差异主要原因是什么？（计量误差、混料、未及时入账、盗窃）如何追责？  下一步一个季度盘点一次
8. 盘点的频率大概多久一次，是否需要财务干预
9. 是否使用条码/RFID？若使用，覆盖哪些环节？（入库打标、上架、拣货、出库扫描）
